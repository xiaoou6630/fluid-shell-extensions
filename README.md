## 流体外壳扩展

## 概述 | Overview

**Fluid Shell Extensions** 是 [Create Big Cannons] 的附属模组，为流体炮弹扩展了一套灵活的自定义效果系统。通过简单的 JSON 配置文件，你可以为任意流体添加丰富的爆炸效果，而无需编写任何代码。

**Fluid Shell Extensions** is an addon mod for [Create Big Cannons] that expands the fluid shell system with a flexible custom effect framework. Through simple JSON configuration files, you can add rich explosive effects to any fluid without writing a single line of code.

## 特点 | Features

- 完全数据驱动 / Fully data-driven
- 多阶段效果 / Multi-stage effects
- 支持多种效果类型 / Multiple effect types
- 软依赖支持 / Soft dependency support
- 详细日志输出 / Detailed logging

## 安装要求 | Requirements

- Minecraft 1.20.1
- Forge 47.0.0+
- Create 6.0.0+
- Create Big Cannons 5.10.2+

## 配置 | Configuration

配置文件位于 config/fluid_shell_extensions/effects/。
